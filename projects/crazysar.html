CrazySAR

CrazySAR is a modular aerial robotic system with decentralized control architecture built on the Crazyflie platform. It consists of a collection of Crazyflie 2.1+ quadrotors connected to each other with fixed rods by spherical joints to create an open chain structure. The decentralized control architecture is modeled off of the Graph-Based Modeling and Control of Articulated Robots project, where each quadrotor is treated as an agent and physically connected agents can share state and control information. The rods are equipped with magents to allow for easy reconfiguration of the structure.

The system demonstrates many of the benefits of a decentralized control architecture, even though it is a single articulated robot. The main features of a decentralized control are:

  • Scalability: The system can support open chain configurations of any size without changing the control law.
  • Flexibility: Any quadrotor agent can take the role (leader/follower) of any other agent in the system. The system also supports heterogeneuos agents with different size and weight.
  • Fault-tolerance: If one agent is behaving erratically or fails completely, the system can recover and regenerate with a new agent.

Open chain configurations

Open chain configurations with the topology of a hub-and-spoke graph and line graph can fly. In both configurations, the leader is in the middle and it sends control instructions to the followers so the forces along the carbon fiber rods are minimized. In the flapper configuration, the 1-hop followers send control instructions again to their followers. Therefore, since each agent relies on the state and control action of one upstream agent, the system is scalable. Also, minimizing the forces along the rods helps decouple agent dynamics across the system.

Flexibility

Here, a simple 2-agent configuration is demonstrated with one Crazyflie replaced with the larger and heavier Crazyflie 2.1 Brushless variant. The control system is able to adjust by increasing the applied forces on the brushless quadrotor to maintain stability.

Split and merge

A CrazySAR's ability to split and merge with other agents on command demonstrates its recoverability. In the demo video, one quadrotor was shut off, which happens when there is an issue with the radio connection or a mechanical failure. When it falls, the magnetic joint is separated, leaving the rest of the configuration unaffected. Using a remote control, the remaining two quadrotors are piloted down to attach with a replacement agent.
